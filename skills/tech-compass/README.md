# Tech Compass — Technology Growth Navigator

> Part of the Clawsight Scene Skill ecosystem

## What It Does

Tech Compass is your actionable growth guide. It takes the self-knowledge from Career Mirror and the positioning from Tech Spectrum, then translates them into a concrete learning roadmap and action plan.

**It answers**: "What exactly should I learn next, and what should I do in the next 30-60-90 days?"

## Key Features

- **Skill Quadrant Matrix**: Maps skills on your-level × market-demand — core assets, priority investments, moats, depriorities
- **AI Skill Layer Assessment (L0-L4)**: Where you stand on the AI value chain
- **Learning Routes**: 2-3 paths with monthly milestones, linked to opportunity windows
- **30-60-90 Day Action Plan**: Immediate, specific, verifiable actions
- **Risk & Adaptation**: What to watch for if things change

## Usage

```
/tech-compass
```

## AI Skill Layers (detailed in `docs/skill-layers.md`)

| Layer | Description |
|-------|------------|
| L0 | AI User — uses AI tools daily |
| L1 | AI Integrator — integrates AI into products |
| L2 | AI Builder — builds AI-powered systems |
| L3 | AI Architect — designs AI-native architectures |
| L4 | AI Definer — defines AI application in a domain |

## Modes

| Mode | When | What You Get |
|------|------|-------------|
| **Enhanced** | Profile + Career Mirror + Tech Spectrum | Precision-guided routes |
| **Rich** | Profile only | Full plan from profile analysis |
| **Lite** | No profile | General recommendations |

## Part of a Chain

```
/career-mirror → /tech-spectrum → /tech-compass
  "Who am I?"     "Where do I stand?"  "Where do I go?"
```

Tech Compass is the action endpoint — consumes both upstream skills for the most targeted guidance.
